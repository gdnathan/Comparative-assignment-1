## requirements

To compile this project, you need `cargo`.

To install, just run
```sh
curl https://sh.rustup.rs -sSf | sh
rustup toolchain install nightly
```

## How to build 

```sh
cargo build
```

## How to run

```sh
cargo run
```

## How to test

```sh
cargo test
```

Once the program is started, type `help` to learn about the different commands.
